This is the old way of using HTML tags for special formatting text

  1. You used to use the "i" tag for italicized text
  2. You used to use the "b" tag for bold text
  3. You used to use the "u" tag for underlined text

This is the new (semantic) approach to doing the above

  1. You now use the "em" tag for italicized text (emphasis).
  2. You now use the "strong" tag for bold text (strong emphasis).
  3. You can use the "ins" tag to get underlined text (apparently intended for text inserted after publication. Perhaps this is where CSS comes in?).
